●프로그램 오류의 종류
1. Syntax Error
문법 오류
해당 언어가 작성한 프로그램을 이해하지 못하는 경우

2. Runtime Error
프로그램 실행 중에 에러 메세지와 함께 갑자기 종료되는 것

3. Sementic Error
프로그램이 에러 없이 실행되지만,
사용자가 기대하지 않은 실행 결과가 나오는 것

●왜 프로그래밍을 배워야 하는가
- 자신의 원하는 일을 하는 프로그램을 만들 수 있음
- 오픈소스 커뮤니티
